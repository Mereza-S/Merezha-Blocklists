# Merezha Blocklists

Офіційні списки блокування компанії "Мережа-С".

## Призначення

Цей репозиторій містить власні списки блокування для:

- азартних ігор;
- VPN;
- проксі;
- анонімайзерів;
- заборонених сервісів;
- соціальних мереж (за потреби);
- інших ресурсів, що використовуються в системі доступу до Інтернету засуджених.

## Структура

```
blocklists/
    gambling.txt
    vpn.txt
    proxy.txt
    social.txt
    blacklist.txt
```

## Формат

Один домен — один рядок.

Наприклад

casino.com
example.bet
example777.net

## Використання

Файли можуть використовуватись у:

- AdGuard Home
- Pi-hole
- MikroTik (після конвертації)
- інших DNS-серверах
